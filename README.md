# ❌⭕ Tic Tac Toe Game

A simple and interactive **Tic Tac Toe game** built using HTML, CSS, and JavaScript.
This project includes game logic, score tracking, and dynamic UI updates.

---

## 🚀 Features

* 🎮 Two-player Tic Tac Toe gameplay
* 🧠 Turn-based logic handling
* 🏆 Score tracking system for both players
* 🔁 Reset game functionality
* 🆕 New game button after win/draw
* 🎨 Clean and responsive UI design
* 💬 Dynamic win message display

---

## 🛠️ Technologies Used

* **HTML5** – Game structure
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Game logic and interactivity

## 🎯 How the Game Works

1. Two players take turns:

   * Player 1 → X
   * Player 2 → O

2. Players click on boxes to place their symbol

3. Game checks for:

   * Winning combinations
   * Draw condition

4. Score updates automatically

5. User can:

   * Reset the board
   * Start a new game
